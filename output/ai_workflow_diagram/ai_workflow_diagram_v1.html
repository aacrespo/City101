What's Different

vs "I used ChatGPT"
01 Version-controlled memory
Every conversation reads and writes to the same repo. Context persists across sessions, not just within a chat window. Git tracks every decision.
02 Specialized agents, not one generalist
5 agents with distinct skills: analyst, cartographer, visualizer, builder, modeler. Each reads different files, produces different outputs, follows different rules.
03 Tool integration, not copy-paste
Agents call APIs directly, run Python scripts, generate maps, verify data quality. The AI operates inside the development environment, not alongside it.
04 Quality gates before production
Nothing reaches datasets/ unverified. Coordinate ranges, null checks, duplicate detection. output/ → verify → datasets/ pipeline enforced.
05 Two-person async collaboration
Andrea and Henna share context through LOCKBOARD.md and CONTEXT.md. Each session starts by reading what the other person did. The repo is the single source of truth.
