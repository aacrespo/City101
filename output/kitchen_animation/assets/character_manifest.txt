REUSABLE ACTIONS (all have fake_user=True)
==========================================

Action_Walk (24 frames / 1 sec)
  - Location Y: 0 -> 2.0 linear movement
  - Cyclic modifier applied

Action_Walk_ShapeKeys (24 frames / 1 sec)
  - Body squash: bounces 0 -> 0.3 -> 0 twice per cycle
  - Pair with Action_Walk on the object

Action_Idle (72 frames / 3 sec)
  - Scale oscillation: +/- 2% sine wave
  - Cyclic modifier applied

Action_Idle_Spark (72 frames / 3 sec)
  - Spark Z rotation: full 360 over 3 seconds

Action_Blink_L / Action_Blink_R (72 frames / 3 sec)
  - Blink at ~frame 48: value 0 -> 1 -> 0 over 4 frames

Action_Spawn (12 frames / 0.5 sec)
  - Scale: 0 -> 1.15 -> 0.95 -> 1.0 (overshoot + settle)

Action_Despawn (10 frames / ~0.4 sec)
  - Scale: 1.0 -> 1.05 -> 0.0 with upward drift
  - Location Z rises during despawn

Action_Working_ShapeKeys (48 frames / 2 sec)
  - Rapid alternating squash/stretch at 0.15 amplitude
  - Cyclic modifier applied

HOW TO USE
==========
1. Append/link the desired Char_ collection into your scene file
2. Assign actions via object.animation_data.action (object actions)
   and object.data.shape_keys.animation_data.action (shape key actions)
3. Never animate directly on the master characters.blend — always work on copies
4. Teammates and subagent can be recolored by swapping the body material
