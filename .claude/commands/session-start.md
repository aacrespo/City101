# Utility: Session Start

Sync and provide a session briefing:

0. Run `git pull` to sync with remote before reading anything.
1. Read `CONTEXT.md` — shared project state.
2. Read `CONTEXT_ANDREA.md` — Andrea's priorities, data gaps, session log.
3. Read `CONTEXT_HENNA.md` — Henna's priorities and session log.
4. Read `LOCKBOARD.md` — who's working on what right now.

Then summarize:
- **Current state**: what phase the project is in, what happened last session
- **Active work**: what Andrea and Henna are each doing
- **Blocked items**: anything stuck
- **Recommended next steps**: what to work on this session

Do NOT modify any files. This is read-only.
