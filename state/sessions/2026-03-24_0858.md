## Key decisions made this session
1. **Characters are blocky voxel Clawds** — not smooth blobs
2. **8 legs** (octopus), **2 stretchy arms** (rubber-hose)
3. **Teammates = same size as personas**, parent's color + role prop (not different colors)
4. **Subagents = 65% scale**, translucent (less context = smaller)
5. **Pixel art style**: render 480x270 → ffmpeg nearest-neighbor upscale to 1080p
6. **Visual review mandatory** — dedicated reviewer checks renders, not just code execution
7. **Arms have full range of motion** — can reach up, wave, bend, not just horizontal

## Blockers
- `bpy.ops.object.transform_apply(scale=True)` in Blender 3.3.9 defaults to applying ALL transforms (location too) — must use explicit `location=False, rotation=False, scale=True`
- Bone parenting via MCP is unreliable — use direct object parenting instead
- Need Agent Teams (not subagents) for coordinated multi-room builds
